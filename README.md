### Quick Manual

##### LShift + CMD ⌘  + 
 
   **H** left half  
   **L** right half  
   **B** left down quarter  
   **.** right down quarter  
   **O** right up quarter  
   **Y** left up quarter  
   **C** left third  
   **V** left two thirds  
   **M** right two thirds  
   **,** right third  
   **J** bottom half  

##### LAlt + CMD ⌘ +
   
   **K** upper half

##### LCtrl ^ + LAlt + CMD ⌘ +

  **←** moves window to left screen
  **→** moves window to right screen
